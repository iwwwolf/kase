Частным инвесторам

Цели инвестирования в фондовую биржу могут быть различные на разных рынках и личных потребностях инвестора. Казахстанская фондовая биржа функционирует в виде электронной торговой системы, которая проводит торги ежедневно по рабочим дням в течении большей части дня образуя торговый день.

торговый день

Казахстанская фондовая биржа функционирует в виде электронной торговой системы, которая проводит торги ежедневно по рабочим дням в течении большей части дня образуя торговый день. Содержание и продолжительность торгового дня устанавливается регламентом торгового дня по операциям купли-продажи иностранных валют и ценных бумаг.

В основу механизма торговли в ходе Торговой сессии заложен принцип “Order driven market” – рынок конкурирующих между собой заявок, при котором сделка заключается автоматически при пересечении условий во встречных анонимных заявках.

Биржа использует следующие методы проведения торгов:

  • метод непрерывного встречного аукциона
  • метод заключения прямых сделок
  • метод фиксинга;
  • метод франкфуртских торгов;
  • метод специализированных торгов;
  • метод английских торгов
  • метод голландских торгов
  • метод подписки
Методы непрерывного встречного аукциона и подписки используются только при проведении торгов в торговой системе

прямой доступ DMA

Кроме того на KASE существует возможность удаленно подключиться к торговой системе с использованием технологии прямого доступа DMA. Технология DMA позволяет клиентам члена Биржи передавать заказ в электронном виде напрямую в программно-технический комплекс Биржи, при этом проверка и подтверждение членом Биржи данного клиентского заказа осуществляется в автоматическом режиме.

Схема DMA

Действующая технология прямого доступа DMA на KASE реализует схему DMA посредством и использования следующих каналов:

Терминал STrade/MTrade
Клиент Брокера, используя терминал STrade/ MTrade, разработанный eTrade.kz, может создавать и передавать транзитные клиентские заказы на брокерский терминал торговой системы Биржи
Терминал KASEe
Клиент Брокера, используя терминал торговой системы Биржи, разработанный Биржей, может создавать и передавать транзитные клиентские заказы на брокерский терминал торговой системы Биржи
FIX-шлюз
Клиент Брокера, используя продукт, разработанный Брокером или независимой софтверной компанией, может создавать и передавать клиентские заказы Брокеру, которые передаются в торговую систему Биржи, посредством системы управления заявками Брокера через FIX-шлюз Биржи.
