Полезные ссылки
  • Для работы с данным разделом вам потребуется интернет-браузер с поддержкой JavaScript. Таковым является практически любой современный браузер, за исключением тех, в которых JavaScript отключен по умолчанию. В этом случае Вам нужно включить поддержку JavaScript в настройках Вашего браузера.
  • Для просмотра определенного рынка или набора выберите его наименование из предлагаемого перечня.
  • Далее выберите необходимые инструменты из списка, используя кнопки "добавить" и "удалить". Можно выделить сразу несколько инструментов, но просмотр возможен по списку, не превышающему 20 инструментов (за исключением набора по Индексу KASE). Можно осуществить поиск инструментов путем набора необходимых символов и нажатием кнопки "Найти". При этом в списке будут найдены инструменты, в кодах которых встречаются введенные пользователем наборы символов вне зависимости от их расположения в наименовании кода.
  • Для удобства выбора нужных полей просмотра можно использовать ссылки "Выбрать все", "Выбрать стандартные", "Убрать все". При выборе стандартных полей осуществляется выбор наиболее часто используемых пользователями полей.
  • Частота обновления страницы просмотра устанавливается пользователем индивидуально.
  • Цветами и стрелками показаны изменения цен финансовых инструментов, произошедшие с момента последнего изменения таких цен в течение торговой сессии. Зеленый цвет означает повышение цены, красный – понижение.
